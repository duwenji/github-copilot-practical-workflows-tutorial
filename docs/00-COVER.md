# GitHub + GitHub Copilot

## 実務ワークフローチュートリアル

日常開発から協業、共有資産管理、`release / hotfix` までを  
**実務シナリオ**で学ぶ日本語教材です。

- 著者: 杜 文吉
- 対象: 開発者 / リード / リポジトリ管理者

### この教材で学べること
- `Issue → Branch → PR → Review → Merge` の基本フロー
- `fork` / `upstream` を使った協業パターン
- `submodule` / `subtree` による共有資産管理
- `release` / `hotfix` とチーム標準化
