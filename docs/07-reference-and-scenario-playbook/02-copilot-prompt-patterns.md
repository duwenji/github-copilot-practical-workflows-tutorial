# Copilot プロンプトパターン集

## 概要

GitHub Copilot Chat でよく使うプロンプトパターンをカテゴリ別に整理したリファレンスです。
要件整理・実装支援・レビュー補助など、場面ごとに参照できます。

## プロンプトパターン一覧

### 要件整理

- 「この Issue を 5 ステップに分解して」
- 「受入条件からテスト観点を列挙して」
- 「この仕様の曖昧な点を洗い出して」

### 実装支援

- 「この関数を読みやすくリファクタして」
- 「例外処理を追加して」
- 「この処理にコメントを付けて」

### レビュー・説明

- 「この変更の PR 説明文を下書きして」
- 「このレビューコメントに対する返信案を出して」
- 「このコードの変更意図を一行で説明して」

### ドキュメント

- 「この関数の JSDoc を書いて」
- 「この設計判断を ADR 形式でまとめて」

## 注意

プロンプトは万能ではないため、**プロジェクト固有ルール** とセットで使います。
Copilot の提案は要件・設計・セキュリティ・テストの観点で必ず確認すること。
